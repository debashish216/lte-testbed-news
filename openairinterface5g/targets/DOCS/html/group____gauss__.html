Detailed Description

This set of routines are used to generate quantized (i.e. fixed-point) Gaussian random noise efficiently. The use of these routines allows for rapid computer simulation of digital communication systems. The method is based on a lookup-table of the quantized normal probability distribution. The routines assume that the continuous-valued Gaussian random-variable, is quantized to bits over the interval where and control the precision and range of the quantization.
